## An easy to use clip editor for gaming

### Features:
- Trim a video clip of your gaming moment
- Either replace or create a new video file
- Optimize the clips to easily send them on chat apps like Discord
  - Automatically compress the clip without changing the resolution/framerate
  - Smart calculation of output file size (make the clip less than 10MB, ideal for no-Nitro Discord)
- Manage your clip folders
- Tag your clips using the tag system (Pentakill, Funny, etc.)
- Automatically delete clips older than a specified period of time
- Sort your clips by date/file size
- Available in English and Polish
